# Company Interview Rounds Reference

This document outlines the typical software engineering interview rounds for major tech companies, which will be integrated into the InterviewGPT platform.

---

## 1. Google
*   **Round 1: Online Assessment (OA) / Phone Screen**
    *   **Type:** Coding / Algorithms
    *   **Duration:** 45 - 60 minutes
    *   **Focus:** 1-2 coding problems, usually covering arrays, strings, or simple graph traversals. Focuses on coding speed and basic data structures.
*   **Round 2: Technical Onsite 1**
    *   **Type:** Coding & DSA
    *   **Duration:** 45 minutes
    *   **Focus:** Algorithmic depth and complexity analysis. Expect advanced trees, heaps, or graphs.
*   **Round 3: Technical Onsite 2**
    *   **Type:** Coding & DSA
    *   **Duration:** 45 minutes
    *   **Focus:** Problem-solving under constraints. Often involves recursion, backtracking, or dynamic programming.
*   **Round 4: Technical Onsite 3 / System Design**
    *   **Type:** System Design or Coding
    *   **Duration:** 45 minutes
    *   **Focus:** Scalability, API design, trade-offs. (Required for Mid-to-Senior roles).
*   **Round 5: Googliness & Leadership**
    *   **Type:** Behavioral
    *   **Duration:** 45 minutes
    *   **Focus:** Ambiguity, teamwork, feedback, alignment with Google's core values.

---

## 2. Amazon
*   **Round 1: Online Assessment (OA)**
    *   **Type:** Coding & Work Style
    *   **Duration:** 90 minutes
    *   **Focus:** 2 coding questions on debugging/DSA, followed by a simulated work scenario and work style assessment.
*   **Round 2: Technical Interview 1**
    *   **Type:** Coding & Leadership Principles
    *   **Duration:** 60 minutes
    *   **Focus:** Data structures and algorithms (arrays, trees, maps) + 15-20 minutes of Behavioral questions based on Amazon's 16 Leadership Principles (LPs).
*   **Round 3: Technical Interview 2**
    *   **Type:** Coding, OOP / System Design & LP
    *   **Duration:** 60 minutes
    *   **Focus:** Object-oriented design, system design (for L5+), or coding + LPs.
*   **Round 4: Bar Raiser Round**
    *   **Type:** Deep Technical & LP
    *   **Duration:** 60 minutes
    *   **Focus:** Led by an independent interviewer (Bar Raiser). Focuses heavily on architectural trade-offs, deep system design, and rigorous LP alignment.

---

## 3. Microsoft
*   **Round 1: Codility OA / Phone Screen**
    *   **Type:** Coding & DSA
    *   **Duration:** 60 - 90 minutes
    *   **Focus:** 2-3 coding problems on Codility platform. Tests basic algorithms, edge cases, and correctness.
*   **Round 2: Technical Onsite 1**
    *   **Type:** Coding & DSA
    *   **Duration:** 45 minutes
    *   **Focus:** Core computer science fundamentals (pointers, arrays, linked lists, strings).
*   **Round 3: Technical Onsite 2**
    *   **Type:** Coding & OOP
    *   **Duration:** 45 minutes
    *   **Focus:** Object-Oriented Design (OOD), class relationships, and clean code.
*   **Round 4: Technical Onsite 3 / System Design**
    *   **Type:** System Design
    *   **Duration:** 45 minutes
    *   **Focus:** Large-scale distributed systems, database choices, load balancing, caching.
*   **Round 5: Behavioral & Fit**
    *   **Type:** Behavioral
    *   **Duration:** 45 minutes
    *   **Focus:** Passion for technology, career growth, team dynamics, and conflict resolution.

---

## 4. Meta (Facebook)
*   **Round 1: Technical Phone Screen**
    *   **Type:** Fast-Paced Coding
    *   **Duration:** 45 minutes
    *   **Focus:** 2 coding questions (often from LeetCode Meta tag). Focuses on speed, optimal solution discovery, and bug-free implementation.
*   **Round 2: Onsite Coding 1**
    *   **Type:** Coding & DSA
    *   **Duration:** 45 minutes
    *   **Focus:** High-efficiency algorithmic design (Graphs, Trees, Arrays).
*   **Round 3: Onsite Coding 2**
    *   **Type:** Coding & DSA
    *   **Duration:** 45 minutes
    *   **Focus:** Complex structures, heaps, sliding windows, and space-time optimization.
*   **Round 4: Onsite System Design / Product Design**
    *   **Type:** System Design
    *   **Duration:** 45 minutes
    *   **Focus:** Designing a system like Instagram Feed, Facebook Messenger, or WhatsApp. Focuses on API design, database schemas, and scalability.
*   **Round 5: Behavioral Round**
    *   **Type:** Behavioral (Jedi)
    *   **Duration:** 45 minutes
    *   **Focus:** Collaboration, resolving conflicts, empathy, and driving impact.

## 5. Apple
*   **Round 1: Technical Phone Screen**
    *   **Type:** Coding & Fundamentals
    *   **Duration:** 45 - 60 minutes
    *   **Focus:** Low-level fundamentals (concurrency, memory management, OS threads) + coding question.
*   **Round 2: Onsite Coding 1**
    *   **Type:** Algorithmic DSA
    *   **Duration:** 45 minutes
    *   **Focus:** Clean code implementation, testing, and handling edge cases.
*   **Round 3: Onsite Coding 2**
    *   **Type:** Language / Framework Deep Dive
    *   **Duration:** 45 minutes
    *   **Focus:** Deep dive into runtime execution, language internals (C++/Java/Swift), and OOP.
*   **Round 4: Onsite System Design**
    *   **Type:** Architecture
    *   **Duration:** 45 minutes
    *   **Focus:** Distributed databases, caching, hardware constraints, or low-level design.
*   **Round 5: Behavioral & Fit**
    *   **Type:** Cultural Fit
    *   **Duration:** 45 minutes
    *   **Focus:** Apple values, attention to detail, work ethic, and cross-functional communication.

## 6. Netflix
*   **Round 1: Initial Technical Screen**
    *   **Type:** Coding & Architecture
    *   **Duration:** 60 minutes
    *   **Focus:** Real-world coding problem + high-level system design discussion.
*   **Round 2: Onsite Tech 1**
    *   **Type:** DSA & Algorithms
    *   **Duration:** 45 minutes
    *   **Focus:** String parsing, graph traversal, and sorting.
*   **Round 3: Onsite Tech 2**
    *   **Type:** System Design & Scalability
    *   **Duration:** 45 minutes
    *   **Focus:** Heavy traffic scaling, caching strategies, fallback mechanisms, and streaming architectures.
*   **Round 4: Culture & Values 1**
    *   **Type:** Cultural Deep Dive
    *   **Duration:** 45 minutes
    *   **Focus:** Discussion on Netflix's unique culture memo ("Freedom & Responsibility", "Keeper Test").
*   **Round 5: Culture & Values 2**
    *   **Type:** Leadership & Fit
    *   **Duration:** 45 minutes
    *   **Focus:** Direct feedback culture, decision-making, and alignment with organizational goals.

## 7. Bloomberg
*   **Round 1: Phone Screen**
    *   **Type:** Coding & Fundamentals
    *   **Duration:** 45 - 60 minutes
    *   **Focus:** Strings, Hash Maps, Linked Lists + C++ or Java concepts (e.g., polymorphism, garbage collection).
*   **Round 2: Onsite Coding 1**
    *   **Type:** DSA / Strings & Stacks
    *   **Duration:** 60 minutes
    *   **Focus:** Multi-threading, custom data structures (like design LRU, custom stock ticker, etc.).
*   **Round 3: Onsite System Design**
    *   **Type:** Design & Architecture
    *   **Duration:** 60 minutes
    *   **Focus:** Real-time data streaming, latency optimization, pub-sub systems, database replication.
*   **Round 4: HR / Hiring Manager Round**
    *   **Type:** Behavioral & Fit
    *   **Duration:** 45 minutes
    *   **Focus:** Adaptability, interest in financial markets, team-player attitude.

## 8. Uber
*   **Round 1: Online Assessment / Screen**
    *   **Type:** Coding / Hard Algorithmic
    *   **Duration:** 60 - 90 minutes
    *   **Focus:** Heavy focus on graphs, 2D arrays, and geometry math.
*   **Round 2: Onsite Coding 1**
    *   **Type:** Graph & Search Algorithms
    *   **Duration:** 60 minutes
    *   **Focus:** Dijkstra, BFS/DFS, pathfinding, or interval scheduling.
*   **Round 3: Onsite Coding 2**
    *   **Type:** Concurrency / System Coding
    *   **Duration:** 60 minutes
    *   **Focus:** Designing thread-safe classes, rate limiters, or concurrent workers.
*   **Round 4: Onsite System Design**
    *   **Type:** Distributed Systems Design
    *   **Duration:** 60 minutes
    *   **Focus:** Location tracking systems, payment gateways, geofencing systems, and dynamic pricing models.
*   **Round 5: Behavioral / Fit**
    *   **Type:** Go-Getter Mindset
    *   **Duration:** 45 minutes
    *   **Focus:** Speed-to-market trade-offs, handling high ownership, and customer obsession.

## 9. Adobe
*   **Round 1: Online Assessment (OA)**
    *   **Type:** Coding & Math/Aptitude
    *   **Duration:** 120 minutes
    *   **Focus:** Multiple choice questions on CS fundamentals + 2 coding questions (usually arrays, math, or DP).
*   **Round 2: Technical Interview 1**
    *   **Type:** Coding & DSA
    *   **Duration:** 45 - 60 minutes
    *   **Focus:** Recursion, linked lists, stacks, trees. Emphasis on writing neat and edge-case-proof code.
*   **Round 3: Technical Interview 2**
    *   **Type:** System Programming / OOP
    *   **Duration:** 45 - 60 minutes
    *   **Focus:** Operating systems (concurrency, processes) + Low-Level Design (LLD) / Object-Oriented Design.
*   **Round 4: Managerial Round**
    *   **Type:** Behavioral & Problem Solving
    *   **Duration:** 45 minutes
    *   **Focus:** Project discussion, system design overview, and company alignment.

## 10. Airbnb
*   **Round 1: Coding Screen**
    *   **Type:** Coding & DSA
    *   **Duration:** 60 minutes
    *   **Focus:** 1-2 coding problems, usually recursion or graph-related (e.g., search autocomplete, Boggle game).
*   **Round 2: Onsite Coding 1**
    *   **Type:** Algorithmic Design
    *   **Duration:** 45 minutes
    *   **Focus:** Clean design, dynamic programming, backtracking, or large-scale string processing.
*   **Round 3: Onsite Coding 2 (Systems/Concurrency)**
    *   **Type:** Real-world API/Low-level implementation
    *   **Duration:** 60 minutes
    *   **Focus:** Code a working component, e.g., thread-safe caching, multi-threaded task runner.
*   **Round 4: Onsite System Design**
    *   **Type:** Large-Scale Architecture
    *   **Duration:** 60 minutes
    *   **Focus:** Booking search engines, payment processing, or reservation inventory management.
*   **Round 5: Core Values**
    *   **Type:** Behavioral / Culture
    *   **Duration:** 45 minutes
    *   **Focus:** Airbnb core values (Be a Host, Champion the Mission, Be a Cereal Entrepreneur).
